Como usamos os dados

Explicação simples de por que o StepGate precisa de certas permissões.

Idiomas: EN · DE · ES · PT

Última atualização: 2026-04-16

Por que é necessário acesso de uso

O StepGate usa estes dados apenas para medir tempo em apps controladas e detectar quando o limite acaba.

Por que são necessários passos/atividade

Os passos atualizam o acesso disponível. Os dados podem aparecer com atraso.

Por que é necessária acessibilidade

Apenas para detetar abertura de apps controladas e mostrar ecrã de limite; não para ler mensagens, texto no ecrã ou teclado.

Os dados saem do dispositivo?

Não. Os dados ficam localmente no dispositivo e não são enviados aos nossos servidores.

O que acontece sem permissões?

O StepGate continua a funcionar, mas funcionalidades relacionadas ficam limitadas.
