Политика конфиденциальности

Как StepGate работает с вашими данными.

Обновлено: 2026-04-16

Введение

StepGate помогает ограничивать доступ к выбранным приложениям и возвращать доступ через активность.

Какие данные использует StepGate

  • Время использования выбранных контролируемых приложений.
  • Шаги/активность из Health Connect и/или доступных источников устройства.
  • Статусы разрешений, нужные для корректной работы функций.

Какие разрешения используются

  • Доступ к статистике приложений — для учета времени в выбранных приложениях.
  • Accessibility Service — для обнаружения запуска выбранных приложений и показа экрана ограничения.
  • Activity Recognition / Health Connect — для чтения шагов.
  • Показ поверх других окон — для экрана ограничения поверх контролируемых приложений.

Как используются данные

  • Расчет базового доступа по недавней истории использования.
  • Обновление дневного доступа после поступления новых данных по шагам.
  • Определение доступного времени в приложениях под контролем.
  • Показ экрана ограничения при исчерпании дневного лимита.

Где хранятся данные

Данные хранятся локально на вашем устройстве. Мы не отправляем usage/steps на наши серверы.

Часть обработки может выполняться системными компонентами Android, которые предоставляют разрешения и API.

Передача третьим лицам

Мы не продаем и не передаем персональные данные третьим лицам.

Сторонние сервисы

StepGate не использует рекламные SDK и стороннюю аналитику.

Что StepGate не читает

  • Сообщения, переписки, email
  • Текст/контент на экране в других приложениях
  • Ввод с клавиатуры и пароли
  • Фото, файлы, контакты

Disclosure для Accessibility

Accessibility используется только для запуска выбранных приложений и показа экрана ограничения после лимита.

StepGate не читает сообщения, не извлекает текст, не перехватывает ввод и не делает посторонние автоматические действия.

Health Connect

При подключении Health Connect приложение запрашивает только чтение шагов для step-based логики.

Хранение данных

Данные остаются на устройстве до удаления данных приложения или деинсталляции.

Как удалить данные

Очистите данные приложения в настройках системы или удалите StepGate.

Дети

StepGate не предназначен для детей младше 13 лет.
